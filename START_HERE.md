## 🎯 The 6 Statuses

1. **Applied** (Slate) - Initial submission
2. **Test Completed** (Purple) - Assessment done
3. **HR Round** (Orange) - Under HR review
4. **HOD Approved** (Cyan) - Department approved (Pursuing only)
5. **Selected** (Green) - Final approval
6. **Rejected** (Red) - Not successful

---

## 🔄 Workflows

**Pursuing Students:**
```
Applied → Test Completed → HR Round → HOD Approved → Selected
```

**Passed Out Students:**
```
Applied → Test Completed → HR Round → Selected
```

## 🎨 Visual Indicators

- **✓** = Completed stage (green checkmark)
- **●** = Current stage (pulsing colored circle)
