#  Internship Management Platform (IMP)

## Project Overview

The **Internship Management Platform (IMP)** is a web-based application designed to manage the complete internship lifecycle — from student application to final certification.

It provides:

* Structured internship workflow
* Real-time progress tracking
* Role-based dashboards
* Transparent evaluation system

---

##  Objectives

* Streamline internship hiring process
* Track student daily progress
* Enable collaboration between HR, mentors, and students
* Automate evaluation and certification

---

##  Installation & Setup

### 1. Clone Repository

```
git clone https://github.com/your-username/internship-management-system.git
```

### 2. Move to XAMPP

Place project inside:

```
C:\xampp\htdocs\
```

### 3. Start Server

* Start Apache
* Start MySQL

### 4. Create Database

Open:

```
http://localhost/phpmyadmin
```

Run:

```sql
CREATE DATABASE internship_db;
```

---

##  Run Project

```
http://localhost/internship-management-system
```
